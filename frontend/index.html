📡 API Endpoints (Postman Reference)

All requests go through the API Gateway on http://localhost:8080. The gateway strips /api and forwards to the Student Service on port 8081.

MethodURLDescriptionBody
GET http://localhost:8080/api/students Get all students —
GET http://localhost:8080/api/students/{id} Get student by ID —
GET http://localhost:8080/api/students/search?q=Alice Search students —
POST http://localhost:8080/api/students Add new student JSON body
PUT http://localhost:8080/api/students/{id} Update student JSON body
DELETE http://localhost:8080/api/students/{id} Delete student —

Sample POST/PUT body:
{ "name": "John Doe", "age": 21, "email": "john@example.com", "course": "Computer Science" }
